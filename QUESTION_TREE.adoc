== Q1: What problem does this bounded context solve, and for whom?

=== Q1.1: Product identity — what is this bounded context?
[ANSWERED]
Evidence: pyproject.toml:5, pyproject.toml:143-144, README.md:20-22, vibe/core/skills/builtins/vibe.py:5-8
Vibe is a "Minimal CLI coding agent by Mistral" — a terminal coding
assistant powered by Mistral models. It ships two entry points:
`vibe` (interactive Textual TUI / programmatic mode) and `vibe-acp`
(an Agent Client Protocol server for editors/IDEs).
