# Demo App

A simple command-line application built for demonstration purposes.

## Features

- Greet users by name
- Add two numbers
- Multiply two numbers
- Simple interactive menu

## Requirements

- Python 3.6+

## Installation

No special installation required. Just run the script directly.

## Usage

```bash
cd demo_app
python main.py
```

## Running the app

1. Navigate to the demo_app directory
2. Run `python main.py`
3. Follow the on-screen instructions

## Example

```
=== Demo App ===
1. Greet me
2. Add numbers
3. Multiply numbers
4. Exit

Enter your choice (1-4): 1
Enter your name: John
Hello, John! Welcome to the demo app.
```
