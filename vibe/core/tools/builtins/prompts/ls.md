Use the ls tool to see the contents of a directory.

## When to use ls vs glob vs grep

- **ls**: See what's in a specific directory (files, subdirectories, sizes)
- **glob**: Find files by name pattern recursively (`**/*.py`)
- **grep**: Search file contents for text or regex

## Tips

- Shows immediate contents only (not recursive)
- Directories are listed first, then files
- File sizes are shown in bytes
- Use glob for recursive file discovery instead
